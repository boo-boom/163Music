# 网易云音乐

> （个人学习日志）vue2.0开发音乐webapp

### 技术栈
* vue2.0
* vuex
* vue-router
* axios
* less
* vue-cli webpack template

### 安装运行

``` bash
# 安装依赖
npm install

# 启动服务
npm run dev

# 打包
npm run build
```
